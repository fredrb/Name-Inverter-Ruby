# Name Inverter

This a version of the name inverter kata using [TCR](https://medium.com/@kentbeck_7670/test-commit-revert-870bbd756864). 

Examples:
- "John" -> "John"
- "John Doe" -> "Doe, John"
- "Mr. John Doe" -> "Mr. Doe, John"
- "Mrs. Jane Doe" -> "Mrs. Doe, Jane"
- "John Doe III" -> "Doe, John III"
- "Mrs. Jane Doe IV Phd." -> "Mrs. Doe, Jane IV Phd."
